# To-Do List Manager (Python CLI)

A simple command-line To-Do List Manager built using Python.  
It allows users to add, view, complete, and delete tasks, with all data stored in a JSON file for persistence.

---

## Features

- Add new tasks with description, tags, and due date  
- View all tasks in a formatted table  
- Mark tasks as completed  
- Delete tasks by ID  
- Stores tasks in a JSON file  
- Handles invalid inputs and file errors safely  

---

## Libraries Used

- json  
- os  
- datetime  

---

2. Clone or download this repository
